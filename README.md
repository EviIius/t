# Power BI Landing Page Animation

A static recreation of the purple Power BI HTML landing page animation from the TikTok reference. It focuses on the dashboard itself: a dark purple report canvas, centered "Power BI Landing Pages" copy, feature pills, subtle grid/light motion, looping translucent bubble choreography, and compound bubbles that split apart and absorb back together.

## Preview

Open `index.html` directly, or run a local static server:

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

Then visit `http://127.0.0.1:4173/`.
